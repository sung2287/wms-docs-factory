WMS Docs Factory

이 레포는 문서 전용 레포입니다.
코드는 포함하지 않으며, 작성된 문서를 코드 레포로 전달하는 역할만 수행합니다.

----------------------------------------
사용 명령어
----------------------------------------

1) main 최신화
- 실행: npm run sync
- 동작: main 브랜치로 이동 후, 원격 main 최신 내용 pull

2) PRD UI import
- 실행: npm run import
- 동작: ./bin/import_prd_ui.sh 실행 후 PRD 문서 자동 반영

3) 문서 완료 후 main 반영 (권장)
- 실행: npm run publish
- 동작: main 체크아웃 → 최신화 → 변경사항 add → 커밋 → main으로 push

4) 현재 브랜치 그대로 푸시
- 실행: npm run push:all
- 동작: 변경사항 add → 커밋 → 현재 브랜치 push

----------------------------------------
기본 작업 흐름
----------------------------------------

npm run sync
npm run import
npm run publish

----------------------------------------
주의사항
----------------------------------------

- 이 레포는 문서 전용입니다.
- 코드 파일을 추가하지 마십시오.
- main 브랜치는 공식 문서 이력 저장소입니다.
